A group or collection of things is called a set. Each thing in a set is called a "member" or "element"

The set of is written in { }

Eg. "the set of languages" can be written as {languages}

A set is normally written out in full so all of its elements can be seen

eg. "the set of the factors of 6" can be written as {1, 2, 3, 6}

The order of the elements in a set doesn't matter.

∈ means "is an element of"

Eg. spotify is an element of the set of apps can be written as

spotify ∈ {apps}

alone is an element of the set of songs can be written as

alone ∈ {songs}

∉ means "is NOT an element of"

Eg. bye is not an element of the set of greetings can be written as

bye ∉ {greetings}

Sets are sometimes given capital letter names to indentify them, eg.

A = {apps}

S = {songs}

F = {files}, etc.
