A subset is a set which is a small part of another set

Eg. {prime number} is a subset of {numbers}

{pop songs} is a subset of {songs}

The sign ⊂ means "is a subset of"

Eg. {cs teachers} ⊂ {teachers}

{black cars} ⊂ {cars}

Listing all the subsets of a set

Eg. (1) The subsets of {apple, banana} are {apple, banana}, {apple}, {banana}, Ø

Eg. (2) The subsets of {1, 3, 5} are {1, 3, 5}, {1, 3}, {1, 5}, {3, 5}, {1}, {3}, {5}

REMEMBER

(1) A set is a subset of itself

(2) An empty set is a subset of any other set

(3) Members of a set may be written in any order

⊃ means "contains as a subset"

⊄ means "is not a subset of"

⊅ means "does not contain as a subset"

Two or more different sets may have one or more members in common. The shared set which contains these members is called the intersection of the original sets

The intersection is written ∩ and is often called cap

Eg. Set L = {a, b, c, d, e, f}

Set V = {a, e, i, o, u}

The memebers a and e are members of both sets so {a, e} is the intersection of sets L and V (L cap V).

Set L ∩ V = {a, e}

In a Venn diagram, intersection is shown by overlapping.

The union of two (or more) sets is the set containing all the members of both (or all) sets

Union is written ∪ and is often caled cup

Eg. P = {Thames, Tay, Tees, Trent, Tweed}

Q = {Clyde, Tay, Tweed, Dee, Forth}

The union of sets P and Q is

P ∪ Q = {Thames, Trent, Tees, Tay, Tweed, Clyde, Dee, Forth}

NOTE Each member of the union should be listed only once. Don't repeat any members

Union of three sets is written A ∪ B ∪ C, etc.
