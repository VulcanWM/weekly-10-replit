A Venn diagram is a way of showing sets in picture form. Each set is drawn as a rough circle, ring or egg shape, like below.

The space inside a Venn Diagram is called a region.

An empty set is a set which contains no elements. It is called a Null set or a Void set.

eg. If set A is the set of six-cornered pentagons

A = { } OR A = Ø

The number of elements in a set is written as n followed by the name of the set in ordinary brackets.

Eg. If set B = {hi, hello, namaste} the number of elements in the set is 5, so it can be written as

n(B) = 3

If set E = {5}

n(E) = 1

The number of elements of an empty set is 0.

Eg. If set H = Ø, n(H) = 0
